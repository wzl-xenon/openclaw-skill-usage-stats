# OpenClaw 用量统计技能

## 概述
本技能用于统计OpenClaw运行期间的使用数据，包括请求次数、Token消耗量、会话明细等，支持自定义时间范围查询。

## 核心功能
1. 统计指定时间范围内的总请求/消息次数
2. 细分维度统计：用户输入、助手回复、工具调用、工具返回结果分类计数
3. 自动分类校验：确保分类总和与总消息数100%一致
4. 按天使用趋势统计，展示每日用量明细
5. 查询Token总消耗量（若有统计数据）
6. 支持生成可视化统计图表，可同时输出24h/7d/30d三个时间维度的图表
7. 支持时间范围：过去24小时（默认）、过去7天、过去30天

## 使用方法
### 1. 仅获取文本统计数据
运行统计脚本获取数据：
```bash
python3 scripts/get_usage_stats.py [时间范围参数]
```

### 2. 获取文本统计 + 生成可视化图表
生成三个时间维度（24h/7d/30d）的统计图表：
```bash
python3 /root/.openclaw/workspace/generate_all_period_charts.py
```

### 参数说明
| 参数 | 说明 |
|------|------|
| 无参数 | 默认查询过去24小时数据 |
| `24h` | 查询过去24小时数据 |
| `7d` | 查询过去7天数据 |
| `30d` | 查询过去30天数据 |

## 输出规范
直接返回结构化的统计结果，包含：
- 统计的时间范围
- 总消息次数
- 细分分类统计：用户输入次数、助手回复次数、工具调用次数、工具返回结果次数
- 分类校验结果：确保分类总和与总消息数一致
- 总Token消耗量（无数据时说明）
- 按天使用明细：每日的总消息、用户、助手、工具消息数量
- 可视化统计图表（生成图表模式下）：24h/7d/30d三张PNG格式图表

## 脚本说明
- `scripts/get_usage_stats.py`：核心统计脚本，自动读取会话日志统计用量数据，自动过滤备份/锁定文件，统计准确。
- `/root/.openclaw/workspace/generate_all_period_charts.py`：图表生成脚本，自动读取统计数据，生成包含占比饼图、数量柱状图、趋势折线图的全量统计图表。
- `/root/.openclaw/workspace/generate_usage_chart.py`：单时间范围图表生成脚本，可自定义时间范围生成对应统计图表。
